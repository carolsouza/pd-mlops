### Como executar esse projeto
Para esse projeto você precisará ter instalado o pyenv para controlar as versões locais do python ou alguma outra forma de fazer isso.

Usando Virtual Env siga os passos:
- Deixe a versão local do seu Python na versão 3.13.3
- Rode o comando para criar o ambiente virtual:

``
  python -m venv .venv 
``

- Em seguida utilize o ambiente virtual:

``
   .\.venv\Scripts\Activate.ps1
``

- Faça a instalação das libs que estão no arquivo requirements.txt:

``
  pip install -r requirements.txt
``

Agora o notebook do projeto poderá ser executado! 😊
